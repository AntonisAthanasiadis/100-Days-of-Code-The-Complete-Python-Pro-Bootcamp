# 🐍 100 Days of Code: The Complete Python Pro Bootcamp

Welcome to my personal playground for Angela Yu's **100 Days of Code: The Complete Python Pro Bootcamp**! This repository serves as a daily log, code archive, and showcase of my journey from Python foundations to building complex, real-world applications.

---

## 🚀 Repository Goals

The objective of this challenge is to code every single day for 100 consecutive days, exploring various domains of software engineering using Python:

- **Days 1 - 14:** Python Programming Basics (Control Flow, Loops, Functions)
- **Days 15 - 31:** Intermediate Python & Object-Oriented Programming (OOP)
- **Days 32 - 58:** Web Development, GUI Apps, Web Scraping, and APIs
- **Days 59 - 80:** Advanced Python Topics & Automation
- **Days 81 - 100:** Professional Portfolio Projects & Data Science

## 🛠️ Built With

Here are the key technologies, libraries, and frameworks I will be exploring throughout this roadmap:

- **Core Python:** Object-Oriented Programming, Advanced Error & Exception Handling
- **GUI & Games:** `tkinter`, `Turtle Graphics`
- **Data Science & Analytics:** `Pandas`, `NumPy`, `Matplotlib`, `Seaborn`
- **Web Scraping & Automation:** `BeautifulSoup`, `Selenium`, `Requests`
- **Web Development:** `Flask`, `HTML5`, `CSS3`, `SQLAlchemy`, `SQLite`/`PostgreSQL`
